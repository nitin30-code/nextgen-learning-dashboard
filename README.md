# 🚀 Next-Gen Learning Dashboard

A modern, animated student dashboard built with **Next.js App Router, Supabase, Tailwind CSS, and Framer Motion**.

---

## 🌐 Live Demo
https://nextgen-learning-dashboard-ikqr-k7e725y2k.vercel.app/

---

## 🛠 Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Supabase (PostgreSQL)
- Framer Motion
- Lucide React

---

## 📊 Features
- Dark mode bento grid UI
- Dynamic courses from Supabase
- Animated progress bars
- Smooth page load animations (Framer Motion)
- Responsive design (mobile, tablet, desktop)
- Sidebar adaptive layout

---

## 🗄️ Supabase Table: `courses`

- id (uuid)
- title (text)
- progress (integer)
- icon_name (text)
- created_at (timestamp)
